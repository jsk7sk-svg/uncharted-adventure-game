# Uncharted Adventure Game

An interactive text-based adventure game inspired by the Uncharted video game series.

## Overview

This project is a Python-based adventure game that allows players to explore different scenarios, battle enemies, make decisions, and progress through a story-driven experience.

## Features

- Interactive storyline
- Player decision-making
- Combat system
- Save and load functionality
- Health tracking
- Inventory management

## Technologies Used

- Python
- PyCharm
- Object-Oriented Programming (OOP)

## Skills Demonstrated

- Software Development
- Problem Solving
- Game Logic Design
- Debugging
- Python Programming
